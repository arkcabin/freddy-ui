# Announcement Blocks

## References
- [Navbar Gallery: Announcement Bars](https://www.navbar.gallery/type/announcement)

## Design Guidelines
- **Height**: Should be thin (typ. 32-48px).
- **Position**: Usually placed at the very top of the viewport (`fixed top-0` or `relative` above the main header).
- **Features**: 
  - Call to Action (Link/Button)
  - Close/Dismiss Button
  - Mobile Responsiveness (truncate text if needed)
  - Theme Awareness (Dark/Light mode support)
